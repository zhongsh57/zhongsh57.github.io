CSS之难题解决和知识补充

Posted on 2020-12-23 Edited on 2020-12-24 In CSS

这里学习一下CSS 很多的之前没用过的东西

颜色

上来就是一个 RGBA 和 HSLA
RGBA 都比较熟悉了 A === Alpha 透明度啦
HSLA A=== 透明度
其实都是颜色 问题不是很大 和前端的关系不大 但是面试可能问到颜色的其他的函数 这里补充一下

border我不知道的秘密

border其实就是边框 但是我们不知道的是 边框默认是会覆盖掉我们的div标签内部的东西的
这个时候 有一个属性叫做background-clip
顾名思义 这个属性是确定我们的背景是如何进行保留的 三个值

  • padding-box 从padding就开始有背景了 这个时候div的大小是content+padding+border
  • content-box 从content才会有背景 这个时候如果设置padding 那么就是无背景的padding
  • border-box border覆盖背景

所以设置外边框 或者是四周的阴影之类的 直接使用 background-clip 设置为padding-box 然后进行border样式的修改就ok啦

但是有一个缺点就是 我们只能设置一个边框 如果要多个边框？

box-shadow

这个属性是CSS3的 其实他的原理很简单 就是在我们的盒子模型的后面一层 加上一个指定大小的 颜色色块 我们可以设置这个颜色色块的延伸 可以设置模糊度 可以设置这个颜色色块的位置 以及内阴影还是外阴影 我们甚至可以设置许多个阴影
我们只需要设置不同的延伸就行啦 注意必须比前一个大

outline

设置描边 当然是描边框啦 这个是在边框之外的 但是和边框不同的是 边框是占据大小的 但是outline是不占据大小的

至此为止 我们现在给div加上边框有几种办法

  1. 直接加border 但是这个时候要注意 border占据大小
    • border-box
    • padding-box
    • content-box
  2. border外面套 outline outline不占据大小
  3. 直接box-shadow 无偏移 伸张量就是大小
  4. 多层边框就是多个box-shadow
    • 两个边框可以border+outline
